# Accelerated Computing

This repository is a small playground for learning and experimenting with different C++17 implementations, with a focus on performance and hardware acceleration.

The main goal of the project is to provide a simple and organized environment to explore:

- standard C++ implementations
- SIMD-based optimizations
- CUDA implementations for GPU acceleration
- benchmarks comparing different approaches
- accuracy tests to verify that results stay consistent across implementations

## Goals

This project is intended to help understand the differences between:

- regular C++ code
- vectorized SIMD code
- GPU-parallel CUDA code

At the same time, it allows comparing:

- execution time
- resource usage
- result accuracy between implementations

## What the Project Contains

The repository is designed to include:

- multiple implementations of the same algorithm or problem
- benchmarks to measure performance
- unit tests or validation tests for correctness
- a CMake-based build system
- examples that are easy to extend with new algorithms and optimization strategies

## Technologies Used

- C++17
- CMake
- SIMD
- CUDA
- testing and benchmarking frameworks, as needed

## Long-Term Goal

Over time, this repository will grow into a collection of practical examples and experiments for learning and comparing high-performance programming techniques in C++.

---
This project is actively evolving and will be expanded as new ideas, algorithms, and optimization methods are added.
